# Hungry Caterpillar

**Hungry Caterpillar** is a small 2D game built in **Python** 
using the **Pygame** library.

The player controls a cute caterpillar that collects berries and avoids obstacles.  
The game has **5 levels**, where the visuals, berries, and obstacles change, 
while the core rule stays the same.

This is my **second Python project**.

---

## Game Idea

The main character is a small green caterpillar.

In each level, it collects a different type of berry while avoiding dangerous 
obstacles.  
The game was designed as a small fantasy-style adventure with a forest atmosphere,
 glowing fireflies, magical light, and custom transition screens between levels.

---

## Levels

- **Level 1** — strawberry, obstacle: mushroom
- **Level 2** — blackberry, obstacle: bee
- **Level 3** — grapes, obstacle: rose
- **Level 4** — cherry, obstacle: crystal
- **Level 5** — blueberry, obstacle: frog

---

## Features

- Start screen with a **Play** button
- 5 different levels
- Score system
- Lives system
- Level transition screens
- **Game Over** screen
- Win screen
- Final **The End** screen
- Sound effects and music
- Animated glowing fireflies
- Caterpillar blinking after taking damage
- Caterpillar head and tail rotation depending on movement direction

---

## Controls

- **Arrow keys** — move the caterpillar
- **Mouse** — click the **Play** button
- **Any key** — continue through intro, level transition, and ending screens
